Programación de PLC`s

La programación se realiza de acuerdo al tipo de proceso que el cliente o la empresa requiera automatizar, y también que es lo que quiera realizar para automatizar el proceso.

Dentro de ello se pueden realizar aplicaciones de escritorio, diseño de interfaces para el usuario, bases de datos, recolección de datos de una maquina o proceso que se requiera saber o tener el conocimiento adecuado de la máquina, etc.
